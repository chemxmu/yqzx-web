化学化工学院拉曼仪器操作培训通知（ 2019.3 ）

——针对卢嘉锡楼激光光谱洁净室的拉曼光谱仪

培训主要面向校内各学院有频繁（每月至少两次）使用学院拉曼光谱仪需要的但尚无相应独立上机资格的研究生、老师。

这次培训的拉曼仪器包括XploRA、Renishaw、Nanophoton和Witec，每个人学生限报一台。为了受训学生有足够的经验能够做可靠的测试，同时避免光谱洁净室人员过多难以管理，对于每个外组的学生在接受培训前将会被要求承诺都 能承担起你在学期间，本人所在的课题组未来所有的拉曼测试任务 ，所以原则上，一个课题组只接受一个同学培训。

请需要拉曼仪器使用培训的同学于3月1日 24:00前到下面的链接中（ http://raman.xmu.edu.cn/apply/ ）在线填写申请表，务必仔细阅读培训须知并认真详尽的填写申请表各项内容。

仪器管理人员将依据申请表具体的填写内容和实验室所能承担的人员情况有选择的接收培训学生并于3月2日公布受训学生名单。

培训流程的具体时间安排可参考上述链接内的内容。

如有疑问可拨2188044询问刘川老师。
